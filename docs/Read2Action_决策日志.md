## 2026-05-13 · 产品版本路线补充（1.0 / 1.1 / 2.0）

**背景**

- 需在文档中固化 **1.0 之后** 的产品方向，避免与「当前不做」混淆；**1.0** 仍以 **mock + 本地状态稳定上线** 为主。

**结论**
