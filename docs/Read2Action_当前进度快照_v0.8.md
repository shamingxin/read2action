## 4. 当前已确认的 1.0 主链路

```text
首页输入内容
```

## 6. 第九阶段已交付摘要（9.1～9.3 验收口径）

**9.1 / 9.2（与历史快照一致，摘要）**

- **Token 与页面壳**：`r2aPageShell1020`、卡片圆角/阴影、顶栏与 Tab 等；首页 1020 + 内层约 720；`/parsing` LIVE 与全站阴影一致。  
- **交互**：空输入置灰可 Toast；模型 Dropdown；详情「⋯」；首页「+」、结果页自定义卡片 Toast；输入区 focus 仅光标。

**9.3（走查 + 收尾 + 复验）**

- **主流程**：`/` → `/parsing` → `/result` → 保存到项目 → `/projects/sha` → 笔记详情 **可跑通**。  
- **`/parsing`**：**四步**依次展示后跳转 **`/result`**；**取消解析**清除定时器，**无幽灵跳转**。  
- **编辑**：`/result` 与详情顶栏「编辑」→ **Toast「编辑功能暂未开放」**。  
- **Sidebar**：「新增项目」**仅 Toast**；项目 **hover / active / ⋯** 与 **当前项目选中态**、**行高一致**；**icon** 轻量对齐。  
- **构建**：`npm run build` 已通过（以仓库当前提交为准）。
