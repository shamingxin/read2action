1. 先用 mock 数据跑通页面。  
2. 不接真实 AI API（直至产品明确立项接入）。  
3. 不接真实数据库。  
4. 不做登录注册。  
5. 不做移动端专项。  
6. 不扩展 2.0 功能。  
7. 优先还原 Figma 6 个页面（允许已知占位与细差）。  
8. 优先保证主流程可演示、可部署预览。  
9. 技术栈：Next.js / React / TypeScript / Tailwind / shadcn(base-nova) / lucide-react。  
10. 组件化开发，不堆大单文件。
