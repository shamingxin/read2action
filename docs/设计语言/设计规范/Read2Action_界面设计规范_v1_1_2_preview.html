界面设计规范 · v1.1.2 · 1.4-0 校正版 · 预览修复

把零散文本沉淀为可回看的知识。

Read2Action 是一款个人知识沉淀工具——AI 是手段，长期回看才是目的。本规范确立 1.4「产品气质重构 + 核心体验补齐」的 warm-restrained editorial 视觉基准：近白画布、中文衬线点睛、深墨蓝主操作、墨棕 ink accent，单一视觉真值。

01 · 颜色系统

Color Palette

全站仅一个主操作色（深墨蓝）、仅一个 ink accent（墨棕）。语义色与同步状态色严格限定使用场景。Ink accent 仅用于核心观点圆圈数字、知识卡片 tag、文本标记，禁止做按钮填充或大面积背景。

02 · 字体系统

Typography

Sans 用于默认界面（Inter + Noto Sans SC），Serif（Noto Serif SC）仅在笔记标题、知识卡片标题与 tag、核心观点圆圈数字与标题、section eyebrow 这几处点睛——这是 R2A 气质的最大杠杆。Mono（JetBrains Mono）用于原文对照与 metadata 数字。

04 · 卡片

Cards

两种卡片类型：内容容器（A 型，纯白 surface + 8px radius + 24px padding）用于笔记列表、面板和笔记详情内容模块；index card 知识卡片（B 型，canvas-soft 暖底 + 4px radius + 18px/20px padding）用于笔记详情中的知识卡片 section——刻意小圆角强调"嵌在纸上"质感。

CONCEPT · 概念

认知带宽

人在同一时间能处理的信息量有上限。任何工具的设计都应该减少认知负担，而不是把"功能丰富"等同于"价值高"。

PRINCIPLE · 原则

沉淀优先于消费

阅读完成只是开始。能否在三周后回看这条笔记还能获得新感悟，才是判断笔记质量的真正标准。

QUOTE · 引用

关于"工具的人性"

"好的工具不会让你觉得在'使用工具'，而是延伸了你的感官与思维。" —— 来自 Tools for Thought 这本书的笔记摘录。

HYPOTHESIS · 假设

中文衬线的气质杠杆

在 90% 内容是中文的产品里，是否做了中文衬线优化，是用户截图时一眼能感知到的差别——比配色重要得多。

05 · 组件特征

Component Signatures

这一节展示 R2A 视觉识别度最高的几个组件——它们是把规范从"色板 + 字号"具体化为"R2A 的样子"的关键。如果只展示色板和字体，看不出和其他暖色调工具的区别；这些组件才是真正的差异化标志。

一句话总结

深度工作的本质，是在注意力稀缺的时代， 刻意为高质量思考保留不受打扰的时间块。

核心观点

  1. ① 注意力碎片化是深度思考最大的敌人 描述层级比标题浅一级，使用 sans、13.5px、ink-secondary，保持阅读流轻盈。
  2. ② 输入之后的安静整理比输入本身更重要 圆圈数字使用 serif 与 ink accent，标题同样使用衬线来形成笔记详情识别度。

06 · 间距系统

Spacing

编辑性间距（仅限内容阅读区）：在笔记详情、解析结果、长引文等内容阅读区，允许使用 14px · 18px · 22px · 36px · 40px · 44px 等非 8 倍数值来制造 editorial 视觉节奏。 结构区（按钮、卡片、输入框、表单、侧栏 padding、组件内 padding）仍严守 8 倍数 token。 纯 8 倍数会让长文阅读节奏过于机械；少量编辑性间距让长文有"被精心排过版"的呼吸感。

08 · 阴影 / 层级

Elevation

默认仅 hairline 描边，禁止 drop shadow。Soft 与 Elevated 使用多层近透明叠影，阴影色用暖色调 rgba(20, 16, 10, …)，避免在暖底上发"脏"。

Flat
默认卡片、行。仅 1px hairline 描边，无 shadow。绝大多数容器使用这一档。
Level 0
Soft
浮起卡片、toast、dropdown。多层近透明暖色叠影，最深 4px 18px。
Level 1
Elevated
Modal、popover。更深的暖色叠影，最深 12px 32px，制造明显浮起感。

09 · 动效 token

Motion

所有 transition 必须显式声明属性，禁止 transition: all。禁止弹跳 / spring / overshoot 类动效——与"安静知识工具"气质相冲。超过 300ms 的过渡仅允许 modal / 抽屉。

--r2a-duration-fast
80ms
颜色、透明度、文字色微变；按钮按下态 scale 反馈。
--r2a-duration-base
150ms
默认 hover、按钮态变化、border 变化。
--r2a-duration-slow
250ms
卡片位移、内容容器切换、modal 进出、主题切换。
--r2a-ease-out
cubic-bezier(0.4, 0, 0.2, 1)
默认 所有 hover / 状态变化。
--r2a-ease-in-out
cubic-bezier(0.4, 0, 0.6, 1)
双向状态（modal 进出、抽屉、主题切换）。
