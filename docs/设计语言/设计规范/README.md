# Read2Action · 界面设计规范

本目录存放 Read2Action 的 **界面设计规范** 与配套 HTML 视觉预览，与 `docs/核心记录/` 中的项目推进文档分离。

## 当前最新基准（开发依据）

| 文件 | 说明 |
|------|------|
| **`Read2Action_界面设计规范_v1_1_2.md`** | 设计规范正文 · **v1.1.2**（1.4-0 校正版） |
| **`Read2Action_界面设计规范_v1_1_2_preview.html`** | 配套 HTML 视觉预览（token、组件、气质对照） |

**设计语言**：warm-restrained editorial — 近白暖画布、中文衬线点睛、深墨蓝主操作色、ink accent 装饰。

## 使用方式

1. **UI 开发或改版前**，优先读取本目录外层的 **v1.1.2 md** 与 **v1.1.2 preview**。
2. 在浏览器中打开 `Read2Action_界面设计规范_v1_1_2_preview.html`，对照 token 与组件约束。
3. 按规范做小步局部对齐；**规范不替代人工验收**。

## 过程稿（仅追溯，非开发依据）

`过程稿/` 目录存放历史版本、草稿，例如：

- v1.0 设计规范及 preview
- 其他未升格为当前基准的中间稿

**1.4 视觉探索 HTML 预览稿**（全流程 UI 等）请放在 `docs/设计语言/1.4视觉预览html/`，不要放在本目录。

**过程稿仅供版本追溯与对照，不作为当前 UI 开发的执行依据。** 若与 v1.1.2 冲突，以 **v1.1.2** 为准。
