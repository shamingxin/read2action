# Read2Action · 1.4 视觉预览 HTML

本目录专门存放 **1.4 阶段 UI 视觉探索** 的 HTML 预览稿。

## 定位

| 类型 | 目录 | 用途 |
|------|------|------|
| **正式设计规范** | `docs/设计语言/设计规范/` | 当前 UI 开发执行依据（**v1.1.2**） |
| **1.4 视觉预览 HTML** | **本目录** | 全流程 / 单页视觉方案的过程稿，供人工对比与选型 |

## 使用说明

1. 这里存放的是 **1.4 视觉探索 HTML 预览稿**，用于查看主流程页面气质、布局与交互示意。
2. 这些文件是 **视觉方案过程稿**，**不是**正式设计规范，**不能**替代 `Read2Action_界面设计规范_v1_1_2.md`。
3. **正式 UI 开发**仍以 `docs/设计语言/设计规范/Read2Action_界面设计规范_v1_1_2.md` 及配套 preview 为准。
4. 每次生成新版预览时，使用 **`v0` / `v1` / `v2` …** 递增命名（例如 `Read2Action_1.4_A_全流程UI预览_v1.html`），**不要覆盖旧版**，方便横向对比与回溯。

## 命名约定

```text
Read2Action_1.4_A_全流程UI预览_v0.html
Read2Action_1.4_A_全流程UI预览_v1.html
Read2Action_1.4_A_全流程UI预览_v2.html
```

- **`1.4_A`**：对应 1.4 子阶段或预览主题（如全流程 UI）
- **`vN`**：该主题下的第 N 版视觉探索稿（从 `v0` 起）
